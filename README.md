# 项目简介 python156

本项目是一个基于Python的农家乐酒店管理系统，分为前端展示和后端管理两大模块，旨在提供全面的信息管理和交流平台。

## 技术栈

- 后端技术：Python、Django框架
- 前端技术：jQuery
- 数据库：MySQL 5.7.33

## 功能模块

### 前台管理

- 农家乐信息
- 美食信息
- 住宿信息
- 活动信息
- 论坛交流
- 系统公告
- 个人中心

### 后台管理

- 农家乐管理
- 美食信息管理
- 住宿信息管理
- 活动信息管理
- 客房预订管理
- 用户评价管理
- 论坛交流
- 管理员管理
- 系统管理

## 系统角色

- 管理员
- 用户

## 运行环境

- 操作系统：兼容Windows、Linux等常见操作系统
- 后端依赖：Python环境（建议版本3.x）、Django框架、MySQL数据库
- 前端依赖：现代浏览器支持（如Chrome、Firefox等）
